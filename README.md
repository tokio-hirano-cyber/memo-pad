# ①課題名
メモ帳アプリ
## ②課題内容（どんな作品か）
株式の売買履歴をブラウザ上で記録できる「トレーディングメモアプリ」を作成しました。
日付、銘柄名（コード）、売買区分、株価、数量を入力することで、履歴一覧に記録され、株価と数量から金額を自動で計算・表示します。
データはブラウザの localStorage に保存されるため、ページを閉じても履歴が保持されます。
## ③工夫した点・こだわった点
Tailwind CSS を使って、見やすくモダンなデザインにした
入力フォームと履歴一覧の X軸（列）を揃えて、視認性を高めた
「買い」と「売り」で文字色を変え、視覚的に分かりやすく工夫した
localStorage を利用して、ページ更新後もデータが消えないようにした
株価 × 数量 の金額を自動で計算し、手間を減らした
## ④難しかった点・次回トライしたこと（または機能）
フォームと履歴一覧の「横位置（X軸）」を揃える部分
tableタグとgridレイアウトの組み合わせによるレイアウト調整
localStorage のデータ形式をどう扱うかの設計
